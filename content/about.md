## this blog 🥯

### build with

基于 [Nuxt Content](https://content.nuxtjs.org/)，使用了 [TailwindCSS](https://tailwindcss.com/) 以及 [Iconify](https://iconify.design/)，评论区组件基于 [Waline](https://github.com/walinejs/waline) 。部署到了 GitHub Pages。搭建过程详细写在了：[🚀 使用 NuxtJS Content 搭建博客](/posts/build-blog-with-nuxt-content)
