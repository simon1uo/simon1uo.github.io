📚 精选笔记最近更新内容（5月）

又是一个月一度的笔记总汇，在此算是一个月度的学习总结了。最近在看 《JavaScript 现代教程》这个文档，顺手把其中的一些重要的内容跟我之前做的 JavaScript 笔记整合了一下，发现知识缺口还是挺多的。之前跟着视频配套的笔记其实并不全面，很多东西也没有更新到位，所以说 B站的很多视频确实入门很有用，但是更加深入的内容，还得看看最新的参考文档、图书。
